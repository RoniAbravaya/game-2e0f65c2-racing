## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Running the App

```bash
# Start Expo dev server
npx expo start

# Run on Android (with Expo Go app)
npx expo start --android

# Run on iOS (with Expo Go app)
npx expo start --ios

# Run on web
npx expo start --web
```

### Running Tests

```bash
npm test
```

## Development Notes

### Ads & IAP in Development
- Ads and In-App Purchases are **mocked** in development mode
- They only work in production builds with native modules
- To test real ads/IAP, create a development build with EAS

### Creating a Development Build
```bash
# Install EAS CLI
npm install -g eas-cli

# Login to Expo
eas login

# Create development build
eas build --platform android --profile development
```

## Customization

This template is designed to be customized by the AI agent. Key areas for customization:

1. **Game Mechanics**: Modify `GameScreen.tsx` to implement specific game logic
2. **Levels**: Edit `config/levels.ts` to add or modify levels
3. **Styling**: Update colors and themes in screen components
4. **Assets**: Replace placeholder images in `assets/` folder

## Building for Production

```bash
# Build Android APK (for testing)
eas build --platform android --profile preview

# Build Android AAB (for Play Store)
eas build --platform android --profile production

# Submit to Google Play
eas submit --platform android --latest
```
